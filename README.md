## Features

## Tech Stack

**Client:** React, Redux, TailwindCSS

## Run Locally

Clone the project

```bash
  git clone https://github.com/Vinitpal/Rowdy-Store
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
